## 📸 Preview

> 🔗 **[https://deenportfolio.netlify.app](https://deenportfolio.netlify.app)**

A premium, fully responsive personal portfolio built with React, Tailwind CSS, and Framer Motion. Designed to look like a professional SDE portfolio — inspired by Linear, Vercel, and Stripe — not a typical student site.

---

## ✨ Features

- 🎨 **Dark Theme** — Deep `#0A0A0A` background with violet, cyan & pink accents
- 🌊 **Smooth Animations** — Framer Motion scroll reveals, page transitions & hover effects
- ⌨️ **Typewriter Effect** — Animated role cycling in the Hero section
- 📊 **Animated Counters** — Stats count up on load (Projects, Internships, CGPA, Wins)
- 🧩 **Glassmorphism Cards** — Frosted glass UI for skills, achievements & contact
- 🏆 **Trophy Showcase** — Glowing hackathon achievement cards
- 📅 **Timeline** — Animated internship & education timeline
- 🗂️ **Project Filter** — Filter projects by category (Web, IoT, AI, Mobile)
- 📬 **Contact Form** — EmailJS-powered form with success/error toast
- 📱 **Fully Responsive** — Mobile-first design across all breakpoints
- ⬆️ **Back to Top** — Floating scroll-to-top button
- 🔢 **Scroll Progress Bar** — Top-of-page reading indicator

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:
- [Node.js](https://nodejs.org/) (v18 or above)
- npm or yarn

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/Deenprasath/Dp_Portfolio.git

# 2. Navigate into the project
cd Dp_Portfolio

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

The site will be live at `http://localhost:5173`

### Build for Production

```bash
npm run build
```
